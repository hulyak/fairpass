## Troubleshooting

### "Balance still shows 0"
- Wait 30 seconds and refresh
- Check browser console for the faucet link
- Make sure you used the **parachain=1000** link (not the regular faucet)

### "Can't connect wallet"
- Install Polkadot.js extension: https://polkadot.js.org/extension/
- Or use **Demo Mode** to test without wallet
